# Base Container: Graylog2
This provides a default [Graylog2](http://graylog2.org/) setup.  This module is managed with Puppet.

# Configuration
Ports:
* 514 (syslog)
* 3000 (graylog dashboard)
* 9200 (elasticsearch)
* 27017 (mongodb)
* 28017 (mongodb admin)
