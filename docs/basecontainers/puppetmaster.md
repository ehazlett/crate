# Base Container: Puppet Master
This provides a default [Puppet](https://puppetlabs.com/puppet/puppet-open-source/) setup.  This module is managed with Puppet.

Note: this container uses Passenger with Puppet.

# Configuration
Ports:
* 80 (apache2)
* 8140 (puppet)
